# Prerequisites

## Tools and libraries

1. Cmake 
2. clang++ (optional, works with g++ as well)
3. Boost >= 1.55
4. Python 3

## Build

`mkdir build`
`cd build`
`cmake ..`
`make`

## Configs

PCPATH should direct to sources directory
PCBUILDPATH should direct to the builded directory
PYTHONPATH += PCPATH + /python/lib
